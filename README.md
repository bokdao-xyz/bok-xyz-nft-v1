# Bok DAO NFT NFT Contract

## Setup

```
STAGING_ALCHEMY_KEY=
PRIVATE_KEY=
PROD_ALCHEMY_KEY=
ETHERSCAN_KEY=
```

## Get Started

```
npx hardhat run scripts/deploy.ts --network goerli
npx hardhat run scripts/deploy.ts --network mainnet
```

## Get Verify

```
npx complie
npx hardhat verify --network goerli ${contract address}
```
